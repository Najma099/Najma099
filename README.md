## 🛠 Projects

### [Zotion](https://github.com/Najma099/zotion) — Real-Time Collaborative Doc Platform

A backend-heavy Notion-like workspace — multiple users editing the same document simultaneously, without conflicts or data loss.

- CRDT-based editing (Yjs) over WebSockets for real-time sync
- Autosave pipeline: `WebSocket → Y.Doc → Redis queue → batched PostgreSQL writes`
- JWT auth with refresh token rotation and session invalidation
- Multi-tenant workspaces, hierarchical pages, RBAC

`TypeScript` `Next.js` `Node.js` `PostgreSQL` `Prisma` `Redis` `WebSockets` `Yjs`

---

### [Suggestrix](https://github.com/Najma099/Suggestrix) — High-Performance Autocomplete Engine

A complete autocomplete + autocorrect engine built from scratch — no libraries, no shortcuts.

- Trie with per-node top-k caching for `O(L)` retrieval
- Usage-pattern-based adaptive ranking
- Fuzzy autocorrect via Levenshtein distance (DP + DFS)

`TypeScript` `Trie` `Levenshtein DP` `React`
